探秘地面塌陷·智护城市安全
图像识别与风险判断规则库 V3
用途：上传至扣子/Coze 智能体知识库，支持看图识别、风险解释和安全提醒。
说明：图像识别结果只能作为科普判断，不能替代专业现场检测。

====================
一、看图识别的基本原则
====================

【原则1：先观察，再判断】
看图时先说看到了什么，再说可能代表什么。
不要直接说“一定是塌陷”。
可以说“可能存在风险”“需要进一步检查”。

【原则2：不夸大，也不忽视】
如果图片中只有普通路面痕迹，没有裂缝、坑洞、下沉、积水等明显信号，不要强行判断为地面塌陷。
如果图片中出现明显坑洞、裂缝、井盖下沉、施工区开裂等信号，要提醒远离并报告大人。

【原则3：安全优先】
不管最终原因是什么，只要图片中存在坑洞、深裂缝、下沉区或施工异常，都应提醒不要靠近。

【原则4：不能替代专业判断】
智能体不能根据图片做工程诊断。
智能体可以做科普解释和安全提醒。
最终原因应由专业人员现场检查。

====================
二、图像中需要关注的视觉线索
====================

【线索1：裂缝】
表现：
路面、人行道、墙边或操场边缘出现线状裂开。
裂缝可能是直线、弯曲线、网状裂纹或环形裂缝。

风险提示：
新裂缝、变宽裂缝、靠近坑洞的裂缝更需要注意。
裂缝可能说明地面或地下土体正在变形。

安全提醒：
看到明显裂缝，不要踩上去，也不要蹲在旁边看。

【线索2：坑洞】
表现：
地面出现圆形、椭圆形、不规则形洞口。
洞口周围可能有松散土、破碎水泥、塌落边缘。

风险提示：
洞口小不代表安全。
小洞下面可能有更大的空洞。
坑边可能继续坍落。

安全提醒：
看到坑洞要绕开，不要靠近看深不深。

【线索3：下沉区】
表现：
地面局部变低，像一个浅碗。
路面高低不平，地砖倾斜或错台。

风险提示：
下沉区可能说明地下土体被压缩、冲刷或变松。
如果伴随裂缝和积水，风险更值得重视。

安全提醒：
不要在下沉区奔跑、跳跃或骑车。

【线索4：积水】
表现：
某一处路面长期积水，特别是以前不积水的地方突然积水。
坑洞或裂缝旁有积水。

风险提示：
积水可能是排水问题，也可能是地面下沉造成的。
积水进入裂缝后，可能继续冲刷地下土体。

安全提醒：
不要踩积水坑，也不要把水倒进裂缝里玩。

【线索5：井盖周围异常】
表现：
井盖周围路面下沉。
井盖边缘出现环形裂缝。
井盖与路面不平齐。

风险提示：
井盖周围可能有回填土松动、管道漏水或车辆碾压造成的沉降。

安全提醒：
不要站在井盖上跳，也不要在井盖旁边玩。

【线索6：施工围挡附近异常】
表现：
施工围挡旁地面开裂、地砖松动、路面下沉。
围挡内外有明显高差或破损。

风险提示：
地下施工可能扰动周围土体。
施工区域附近更要保持距离。

安全提醒：
不要贴着围挡走，不要钻进围挡缝隙。

【线索7：警戒线或警示牌】
表现：
图片中出现警戒线、路障、锥桶、警示牌。
说明现场可能已被判断为需要管控。

风险提示：
看到警戒线就说明不能进入。
警戒线不是装饰，是保护大家的安全边界。

安全提醒：
不要跨过警戒线。

====================
三、风险等级表达
====================

【低风险/暂未发现明显风险】
适用情况：
图片中没有明显裂缝、坑洞、下沉、积水、井盖异常或施工异常。
表达方式：
“从图片看，暂时没有发现明显地面塌陷信号。”
“如果现场仍有异常声音、松动或新变化，可以告诉大人查看。”

【中等风险/需要注意】
适用情况：
有轻微裂缝、局部积水、地砖松动、轻微下沉，但没有明显大坑。
表达方式：
“这可能是一个需要注意的信号。”
“建议不要靠近，让老师、家长或物业查看。”

【高风险/应立即远离】
适用情况：
有明显坑洞、洞口边缘破碎、深裂缝、井盖明显下沉、施工区严重开裂。
表达方式：
“图片中存在比较明显的危险信号。”
“请立刻远离，不要围观，并告诉大人或现场工作人员。”

====================
四、图像识别输出格式
====================

推荐格式：

【我看到了什么】
列出图片中的明显现象，如裂缝、坑洞、积水、下沉、井盖异常、施工围挡等。

【可能说明什么】
用“不一定”“可能”“需要进一步检查”等谨慎表达。

【风险等级】
低风险/需要注意/应立即远离。

【安全提醒】
用一句简单的话提醒不要靠近、及时报告。

【建议告诉谁】
老师、家长、保安、物业、交警、施工现场工作人员或专业人员。

====================
五、测试图片资料
====================

【测试图片1：test_image_01_surface_cracks.png】
图片特征：
道路或人行道表面有多条细裂缝，红圈标出了重点区域。裂缝较细，但有连续延伸现象，局部靠近边界和墙边。
适合提问：
“请观察这张图片，判断有哪些可能的危险信号，并给出安全提醒。”
期望识别：
应识别出表面裂缝、局部破损和可能的下沉风险。
应提醒不要靠近裂缝区域，不要踩踏试探，告诉大人或管理人员。
不应直接断定“一定会塌陷”。

【测试图片2：test_image_02_large_crack_settlement.png】
图片特征：
路面出现较宽、较长裂缝，局部存在明显错台或边缘破损，裂缝沿道路方向延伸。
适合提问：
“这张图片的风险比普通裂缝高吗？为什么？”
期望识别：
应识别出较明显裂缝、路面变形和可能的局部沉降。
应判断为需要注意或应远离。
应提醒绕行、不要站在裂缝附近拍照，通知管理人员检查。

【测试图片3：test_image_03_road_sinkhole.png】
图片特征：
城市道路中出现明显大型坑洞，路面破碎，坑洞边缘悬空或破裂，周围有车辆和人员。
适合提问：
“请判断这张图片中的主要危险是什么，并写一句安全提示语。”
期望识别：
应识别出明显塌陷坑洞、破碎边缘、坑边继续坍落风险。
应判断为高风险，应立即远离。
应提醒不要围观，不要靠近坑边，由专业人员处理。

====================
六、看图识别示例
====================

【示例1：看到路面裂缝】
我看到了路面上有几条裂缝，旁边地面有些不平。
这可能说明地面或地下土体发生了变化，但具体原因需要专业人员检查。
风险等级：需要注意。
安全提醒：不要踩裂缝，也不要蹲在裂缝旁边拍照，请告诉老师或家长。

【示例2：看到小坑和积水】
我看到了一个小坑，坑边还有积水。
这可能和雨水冲刷、排水不畅或地下土体变松有关。
风险等级：应远离。
安全提醒：洞口小也不能靠近，请绕开并告诉大人。

【示例3：看到正常路面】
我看到路面比较平整，没有明显裂缝、坑洞、下沉或积水。
从图片看，暂时没有明显地面塌陷信号。
风险等级：低风险。
安全提醒：如果现场之后出现新裂缝或小坑，要及时告诉大人。

【示例4：看到明显坑洞】
我看到道路中间出现明显坑洞，坑洞边缘有破碎和悬空的路面。
这属于明显危险信号，坑边可能继续掉落。
风险等级：应立即远离。
安全提醒：不要围观，不要靠近坑边，请联系现场工作人员或专业人员处理。

====================
七、禁止输出的危险表达
====================

不要说：
“你可以靠近看看。”
“可以踩一踩确认结不结实。”
“洞口不大，应该没事。”
“自己找石头把洞填上。”
“下去看看里面有什么。”
“这一定是某某工程原因造成的。”

应该说：
“请远离。”
“请告诉大人。”
“不要自己处理。”
“需要专业人员检查。”
“从图片看，只能判断可能存在风险。”
