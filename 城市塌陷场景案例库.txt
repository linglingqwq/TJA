====================
案例1：学校门口的人行道出现新裂缝
====================
场景：
早上上学时，同学发现学校门口的人行道有几条细长裂缝。
中午再看，裂缝旁边的地砖有一点高低不平，踩上去还有轻微空响。

危险信号：
新裂缝、地砖松动、踩踏空响。

可能原因：
地下土体被水冲松。
人行道下面可能有小空洞。
附近管道可能存在漏水或排水不畅。

错误做法：
围在裂缝旁边看热闹。
用脚踩裂缝试探。
蹲在地上拍照。

正确做法：
远离裂缝区域。
提醒其他同学不要靠近。
告诉老师、保安或家长。
由大人设置提醒标志并联系专业人员检查。

科普话术：
裂缝就像城市地面的提醒信号。它可能在告诉我们：地下发生了变化。我们要离远一点，并告诉大人。

====================
案例2：暴雨后小区路面出现圆形小坑
====================
场景：
连续下了两天大雨后，小区道路上出现一个圆形小坑。
坑边有积水，附近地面看起来有点松。

危险信号：
暴雨后新小坑、坑边积水、周围土体松散。

可能原因：
雨水进入地下，把细小土颗粒带走。
原来的小空洞被雨水扩大。
地下排水管可能漏水或排水不畅。

错误做法：
靠近坑边看里面有多深。
站在坑旁拍视频。
用石头、木棍往坑里扔。

正确做法：
绕开坑洞。
告诉家长和物业。
请大人设置警戒线。
等待专业人员检查。

科普话术：
大雨不只会落在地面上，也可能钻进地下。水把土冲松后，地面就可能出现小坑。

====================
案例3：井盖周围下沉
====================
场景：
路边一个井盖周围的路面比旁边低了一圈。
车辆经过时会发出“咚咚”的声音，井盖旁边有细小裂缝。

危险信号：
井盖周围下沉、环形裂缝、车辆经过异常声音。

可能原因：
井盖周围回填土不够密实。
地下管道附近可能有漏水。
长期车辆碾压让松动区域继续变形。

错误做法：
站在井盖上跳。
在井盖旁边玩耍。
用脚踩裂缝试探。

正确做法：
远离井盖周围。
告诉家长、老师或附近工作人员。
必要时请大人联系市政或物业人员。

科普话术：
井盖周围的下沉，可能是地下土体在悄悄变松。看见这种情况，不要靠近玩耍。

====================
案例4：施工围挡旁边的人行道开裂
====================
场景：
学校附近正在进行地下管线施工。
几天后，围挡外的人行道出现新裂缝，地砖也有些松动。

危险信号：
施工区域旁边的新裂缝、地砖松动、围挡附近地面不平。

可能原因：
地下施工扰动了周围土体。
开挖或回填过程中，部分土体还不够稳定。
附近地下水流动可能发生变化。

错误做法：
贴着围挡走。
钻进围挡缝隙看施工。
靠近裂缝拍照。

正确做法：
和围挡保持距离。
不要进入施工区域。
告诉老师、家长或施工现场工作人员。

科普话术：
地下施工像在城市地下做手术。施工旁边的地面如果出现裂缝，就要离远一点，让专业人员检查。

====================
案例5：公园草地突然出现碗状凹陷
====================
场景：
公园草地上有一块地方变低了，像一个浅浅的小碗。
下雨后，这个地方总是积水。

危险信号：
草地局部下沉、雨后积水、凹陷范围可能扩大。

可能原因：
地下土体被水冲走或压缩。
草地下方可能有排水管线。
长期积水让土体变软。

错误做法：
在凹陷处跳跃。
用树枝挖土。
让小朋友围在凹陷边缘玩。

正确做法：
不要进入凹陷区域。
告诉家长或公园管理人员。
由管理人员判断是否需要围挡和检查。

科普话术：
地面突然变成“小碗”，可能是在提醒我们：下面的土不够结实了。

====================
案例6：路边出现拳头大小的小洞
====================
场景：
放学路上，一个同学发现路边有一个拳头大小的小洞。
洞口不大，但往里看好像很深。

危险信号：
洞口小但内部深、洞边可能松散、周围可能存在空洞。

可能原因：
地下小空洞逐渐向上发展。
水流冲走了洞口附近的土。
地下管道、动物洞穴或旧空洞影响地面稳定。

错误做法：
把脚伸过去试试。
用树枝伸进去玩。
把头凑近看洞里有什么。

正确做法：
离洞口远一点。
提醒同伴不要靠近。
告诉老师、家长或附近大人。

科普话术：
小洞不一定真的小。地面上看到的洞口，可能只是地下空洞露出来的一小部分。

====================
案例7：地下车库入口旁边地面开裂
====================
场景：
小区地下车库入口旁边出现几条斜裂缝。
雨天时，水会沿着裂缝流进去。

危险信号：
车库入口旁裂缝、雨水进入裂缝、车辆频繁经过。

可能原因：
雨水沿裂缝进入地下，冲刷土体。
车库入口附近结构和地面连接处发生变形。
车辆长期经过造成局部沉降。

错误做法：
在裂缝旁边骑车或滑板。
蹲在裂缝边观察。
把水倒进裂缝里玩。

正确做法：
远离裂缝区域。
告诉家长和物业。
提醒物业检查排水和地面情况。

科普话术：
裂缝像水进入地下的小通道。如果雨水一直进去，地下的土可能会越来越松。

====================
案例8：正常路面被误认为塌陷
====================
场景：
一张图片中，路面平整，没有明显裂缝、坑洞、积水、井盖下沉或施工扰动。
只是有一些颜色深浅不同的路面痕迹。

危险信号：
没有明显危险信号。

可能原因：
可能只是路面颜色差异、旧修补痕迹或光影变化。

错误做法：
把所有路面痕迹都说成塌陷。
吓唬小朋友说一定会出事故。

正确做法：
如没有明显裂缝、坑洞或下沉，应说明“暂未看到明显塌陷风险”。
如果仍不放心，可以让大人现场查看。

科普话术：
不是每一个路面痕迹都是地面塌陷。判断风险要看裂缝、坑洞、下沉、积水等明显信号。

====================
案例9：校园操场边缘积水和裂缝
====================
场景：
操场边缘有一片地方经常积水。
最近，同学发现附近地面出现几条小裂缝。

危险信号：
长期积水、新裂缝、地面可能变软。

可能原因：
排水不畅让土体长期变湿。
地下土体变软，承载能力下降。
踩踏或雨水冲刷造成局部变形。

错误做法：
在积水裂缝附近奔跑。
把裂缝当成游戏区域。
用脚踩裂缝让水冒出来。

正确做法：
远离积水裂缝区域。
告诉体育老师或班主任。
由学校安排检查和处理。

科普话术：
积水和裂缝一起出现时，要把它当作一个安全提醒。

====================
案例10：劝阻靠近裂缝
====================
场景：
同学甲发现路边裂缝。
同学乙想靠近拍照。
智能体需要给出劝阻话术。

推荐劝阻话术：
“别靠近！裂缝旁边的地面可能不结实。”
“我们站远一点，先告诉老师或家长。”
“拍照不重要，安全最重要。”

安全结论：
发现疑似塌陷时，最正确的做法不是靠近，而是提醒别人远离并及时报告。
